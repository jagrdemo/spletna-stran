Priprava spletne strani

Za pripravo spletne strani je dobro poznati osnove

  • jezika HTML,
  • jezika CSS,
  • jezika JavaScript.

Gradiva in vodičev je na spletu zelo veliko. Za kratko osvežitev si lahko ogledate posnetke v YouTube kanalu Web Demystified. Bolj pregleden in podroben je vodič MDN Web Docs.

Pri pripravi datotek si lahko pomagamo z urejevalnikom Visual Studio Code. Za predogled nastajajoče spletne strani uporabimo paket Live Server ali HTML Preview.
